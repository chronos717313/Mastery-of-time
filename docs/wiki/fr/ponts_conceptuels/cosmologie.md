# Pont 5 : Cosmologie

## D'ΛCDM Uniforme à l'Expansion Différentielle

### ΛCDM : Univers Uniforme
ΛCDM suppose un Univers **statistiquement uniforme** :
- Même composition partout (Ω_m, Ω_Λ constants)
- Expansion uniforme H(z) = H₀√(Ω_m(1+z)³ + Ω_Λ)
- Pas de dépendance à l'environnement local

### TMT v2.4 : Expansion Différentielle
TMT introduit l'**expansion différentielle** H(z,ρ) :

```
H(z, ρ) = H₀ · √[Ω_m(1+z)³ + Ω_Λ · (1 + 0.2 · (1 - |α|² + |β|²))]
```

- **Dépendance à la densité locale ρ**
- **Vides profonds** : expansion accélérée (H relatif +25%)
- **Amas denses** : expansion ralentie (H relatif -60%)

### Implications Observationnelles
- **Résout la tension Hubble** : vides locaux accélèrent H₀ apparent
- **Prédictions testables** : variation Δd_L selon environnement
- **Compatible CMB** : effets locaux ne perturbent pas les échelles grandes

### Validation
- **SNIa par environnement** : Δd_L = 5-10% prédit, 28-44% observé
- **Tension H₀** : 100% résolue
- **CMB Planck** : compatible à 100%
