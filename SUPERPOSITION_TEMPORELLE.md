## Concept Fondamental

### Proposition

Dans notre référentiel, **deux flèches du temps coexistent en superposition**:
- **Temps conventionnel** (t → +∞, expansion)
- **Temps inversé** (t → -∞, contraction)

Cette superposition crée **trois régimes dynamiques**:
1. **y < c** : Sous-luminale (matière ordinaire)
2. **y = c** : Luminale (photons, frontière)
3. **y > c** : Super-luminale (matière "inversée temporellement")

## Formalisation Mathématique

### 1. Superposition d'États Temporels

En analogie avec la superposition quantique:

```
|Ψ⟩ = α|t⟩ + β|t̄⟩

où:
- |t⟩ : état temps conventionnel (forward)
- |t̄⟩ : état temps inversé (backward)
- |α|² + |β|² = 1 (normalisation)
```

### 2. Distorsion Temporelle Bidirectionnelle

Notre distorsion τ(t) = (t/t₀)^(2/3) devient:

```
τ_total(t) = α² · τ_forward(t) + β² · τ_backward(t)

τ_forward(t) = (t/t₀)^(2/3)         (expansion, y < c)
τ_backward(t) = (t₀/t)^(2/3)        (contraction, y > c)
```

### 3. Métrique Effective

En Relativité Générale, la métrique devient:

```
ds² = -c²(α² - β²)dt² + a²(t)[α² + β²](dx² + dy² + dz²)

où:
- α² - β² : facteur temporel net
- α² + β² = 1 : normalisation spatiale
```

**Cas particuliers:**
- α = 1, β = 0 : Temps purement forward (univers standard)
- α = 0, β = 1 : Temps purement backward (univers contractant)
- α = β = 1/√2 : **Superposition maximale** (point critique y = c)

